# employee-mgmt-system(Angular 14)
## Run Locally
Git Clone
cd root folder

`npm install --save npm-install-global`

##  Run Server

`cd server`

`npx ts-node src/server.ts`

## Run Client

`cd client`

`ng serve -o`

> Install and link mongodb and typescript.
> Run both fronted(client) and backend (server)
